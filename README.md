# 🌦️ Smart City Weather Dashboard

## 📌 Overview

A modern and interactive weather dashboard that provides real-time weather data, 5-day forecasts, and intelligent lifestyle recommendations like **"What to Wear"** based on environmental conditions.

Designed with a focus on **user experience, data visualization, and smart logic**, this project demonstrates strong **frontend + API integration skills**.

---

## 🚀 Features

* Search weather by city name
* Auto-detect user location *(Geolocation API)*
* Real-time weather data *(temperature, humidity, wind, etc.)*
* 5-day weather forecast
* Smart recommendations *(What to Wear system)*
* Filter & sort forecast data *(using HOFs)*
* Dynamic UI based on temperature conditions
* Dark / Light mode toggle *(with persistence)*
* Global city comparison dashboard

---

## 🛠️ Tech Stack

* **HTML5**
* **Tailwind CSS**
* **JavaScript (ES6+)**
* **OpenWeatherMap API**

---

## ▶️ How It Works

* The app fetches real-time weather data using the **OpenWeatherMap API**
* Uses **Geolocation API** to detect the user’s location
* Applies **JavaScript logic** to generate smart recommendations
* Uses **array methods (`map`, `filter`, `sort`)** for data handling
* Stores theme preference using **localStorage**
